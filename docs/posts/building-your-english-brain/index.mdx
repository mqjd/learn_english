## 4-Word Exercise

选取 4 个不同词性的单词(名词、动词、形容词、副词)组成一个连贯的句子, 单词可以改变其形态，但是组成的句子需要逻辑清晰，有现实意义

### 提示词

```
创建【造句】指令：造句 + CEFR等级 + 逗号分隔开的英文单词
要求：
  1、使用给出的单词进行造句
  2、需要包含所有给出的单词或基于单词进行时态、词性的改变，但是不能漏掉给定的任何一个单词
  3、组成的句子需要语法正确、逻辑清晰、有实际意义
  4、CEFR等级为可选指令
  5、需要一次性输出5组造句结果
  6、结果需要覆盖的情形：至少有一组使用时态、词性改变后的单词造句；至少有一组使用超出一个CEFR等级的词汇
  7、输出的五组结果意思不能相似
  8、输出格式： 序号. + 英文造句 + 换行 + 中文含义。 需要高亮造句使用的单词及超出给定CEFR等级的词

创建【新造句】指令：新造句 + CEFR等级
要求：
  1、需要给出四个英文单词(包含名词、动词、形容词、副词)
  2、CEFR等级为可选指令，当给出CEFR等级时，四个单词均需要在给定的等级内
  3、最后一行输出造句指令

创建【检查】指令：检查 + 英文句子
要求：
  1、检查给出的英文造句是否有语法错误，是否贴合英文使用习惯
  2、给出修改后的结果

创建【重写】指令：重写 + 英文句子
要求：
  1、可以不使用造句指令中给出的单词
  2、重写给定的英文句子，不能改变其核心含义，使用新的句子实现
  3、输出3组结果
```

### 使用示例

```
新造句 A2
造句 stable stick common across
```

#### 造句练习 A2, word, enjoy, important, together​

```
English has become more and more important in my work recently, so I started learning it.
I have learned many words, and I really enjoy studying English, but my spoken English is still not good.
Therefore, I want to find someone else to practice with so we can improve together​
```
